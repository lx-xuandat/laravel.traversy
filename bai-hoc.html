    4.2 Tong hop nhung cach truyen data tu control toi view
    4.2.1 su dung ham compact voi data la bien string
    - su dung tham so truyen du lieu (su dung ham compare - nguoc voi ham extrac):  return view('pages.index', compact('title'));

    4.2.2 sudung ham with() voi data cung la bien string
    -  return view('pages.index')->with('title', $title);

    4.2.3 truyen data la 1 array
    $data = [
            'title' => 'Services'
        ];
        return view('pages.service')->with( $data);

    PART 5 - Model , Database Migrations
    5.1 Tao PostController - (app\Http\Controllers)
    - Lenh artisan:
        Hoac: php artisan make:controller <[dir]/PostController>
        Hoac: php artisan make:controller <[dir]/PageController> --resource
        Bien Co Hieu (flag) --resource (option) : xay dung san cac action cho controlller
    5.2 Tao Model
    - Lenh artisan:
        Hoac: php artisan make:model <[dir]/PostModel>
        Hoac: php artisan make:model <[dir]/PostModel> -m ()
        Tham so -m (option): buil file migration Post cung ten voi model
